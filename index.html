Playlists For The Lazy &
Tasktrackr

Playlists For The Lazy and Tasktracker are two of my favorite weekend projects. Their UI/UX is a bit rough as they were never intended for public consumption, but rather as personal tools to help me throughout my day.

Playlists For The Lazy is a Youtube player, built with ReactJs, which parses Reddit and creates playlists based on the videos posted in a particular subreddit. It also archives all playlists and allows users to favorite videos and created custom playlists.

Tasktracker is a TODO list with a twist. It uses a knapsack algorithm and informs the user which tasks they should complete for any given amount of free time.
